# TCM-SAN：中医肺癌辨证智能模型

本仓库为论文：

**《基于机器学习的中医肺癌辨证智能模型的构建研究》**

提供相关源码及处理后的实验数据，用于中医肺癌证型分类模型的复现与方法验证。

---
## 一、数据说明

- 医案数量：1083例肺癌中医医案  
- 证型数量：8种中医证型  
- 特征构成：
  - 症状特征
  - 面色特征
  - 舌象特征
  - 脉象特征  

所有四诊信息均经过术语规范化处理，并采用二值化方式进行编码（有/无）。

⚠️ **数据可获取性说明**  
由于医案数据涉及伦理与隐私问题，原始临床医案文本不予公开。  
本仓库仅提供脱敏后的结构化特征数据及示例数据，用于模型训练与实验复现，不涉及任何可识别患者身份的信息。

---

## 二、模型简介

本文构建并比较了以下五种模型：

- 随机森林（Random Forest）
- 支持向量机（SVM）
- 多层感知机（MLP）
- CatBoost
- TCM-SAN 模型

其中，TCM-SAN 模型融合中医“四诊合参”的理论特点，主要包括：
- 特征注意力机制
- SE 通道注意力模块
- 四诊分组随机失活模块（Four-Diagnosis Group Dropout）
- 残差多层感知机编码器
- Focal Loss 分类器
该模型能够突出关键四诊特征，提高对证型分类任务的预测性能与稳定性。
---

## 三、实验设置
- 数据集划分：训练集 70%，测试集 30%  
- 评价方式：五折交叉验证  
- 评价指标：
  - 准确率（Accuracy）
  - 精确率（Precision）
  - 召回率（Recall）
  - F1 值
  - AUC 值
